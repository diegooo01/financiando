# Financiando Ando

Bot de finanzas personales por WhatsApp. Le escribes *"almuerzo 20"* y registra,
categoriza y resume tus gastos, sin abrir ninguna app.

Proyecto de portafolio: el objetivo no es solo que funcione, sino demostrar
criterio de ingeniería (seguridad, tests, decisiones documentadas).
